### Задание 1

Выполнено развертывание инфраструктуры с помощью Terraform. Созданы VPC, публичная и приватная подсети, маршрутная таблица, NAT-инстанс и две виртуальные машины.  
После завершения отображены выходные параметры с IP-адресами созданных ресурсов.  
![terraform 1](https://github.com/ANDREYTOLOGY/terraform-hw/blob/main/img/clopro1-1.png)  

Выполнено подключение по SSH к публичной виртуальной машине. Проверен доступ в Интернет с помощью ping 8.8.8.8 и определён внешний IP-адрес командой curl ifconfig.co.  

![terraform 1](https://github.com/ANDREYTOLOGY/terraform-hw/blob/main/img/clopro1-2.png)  

Выполнено подключение к приватной виртуальной машине через публичную ВМ с использованием SSH Jump Host. Проверен доступ в Интернет командой ping 8.8.8.8, что подтверждает корректную работу NAT-инстанса и маршрутизации.  
Дополнительно выполнена команда curl ifconfig.me, определившая внешний IP NAT-инстанса.  
![terraform 1](https://github.com/ANDREYTOLOGY/terraform-hw/blob/main/img/clopro1-3.png)
